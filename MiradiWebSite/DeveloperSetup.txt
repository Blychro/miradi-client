Install ruby
Install rails
Install MySQL

cd $CVSROOT/cmp-eam/MiradiWebSite/

Create the log dir for the server to use.

mkdir logdir

Start the Miradi website (uses lighttpd by default)

script/server 

If lighttpd is not installed or is not working corrctly, use webrick

script/server webrick

The website should now be working on localhost.

http://localhost:3000

MySQL Setup

Create the user miradi and the database miradi_development

Login as mysql's root user, and using the mysql database

$ mysql --user=root mysql

Create the MySQL user 'miradi'

mysql> GRANT ALL PRIVILEGES ON *.* TO 'miradi'@'localhost';

Query OK, 0 rows affected (0.00 sec)

mysql> quit
Bye

Login as miradi;

$ mysql --user=miradi

Create the miradi_development database
 
mysql> CREATE DATABASE miradi_development;

Query OK, 1 row affected (0.00 sec)

mysql> quit
Bye

Now prepare the newly created database.

rake migrate

Restart the miradi.org rails app

ctrl c (in the terminal the server is running in)

script/server [webrick | lighttpd]
