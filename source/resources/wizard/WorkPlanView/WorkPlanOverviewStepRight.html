Hints for the Work Plan View

You can specify which Conceptual Model or Results Chain to display in the Work Plan tab by selecting from the Diagrams - include data from and the Display this selection drop down choosers.

You can also specify whether to display only Action related rows (Strategies and Activities) or Monitoring related rows (Strategies and Monitoring Activities) by clicking the <Customize Table...> button and making a selection in the Include Data From drop down menu. The project totals values will only reflect the data that is being displayed.

Click on the Work Plan tab to see an overview of your plan. In the tree, clicking on the + and - symbols enables you to expand or collapse different parts of the hierarchy. Click the <Expand To...> button to select the branches of the tree to display. To display the Assignment and Expense rows, click the <Customize Table...> button, and check the appropriate Rows check boxes. Items in the table are based on your results chains and will include the strategies, activities (regular or monitoring) and tasks.

Expenses are estimated in the default project currency. To set the currency, go to the <Settings> tab in the Work Plan View (or go to Edit->Preferences, and select the Planning tab). Select the <Financial Settings> sidetab to adjust the currency type and the number of decimal places displayed.

Hint: The values in the Who Work Assignments column also roll up from lower levels in the hierarchy so that it is easy to see all the resources that will be required for a given strategy or indicator.

Hint: The values in the Timeline column are entered from the <Timeline> sidetab in the sub-panel of an action or from the cell popup, and are independent of the entries in the Work Plan table.

Hint: The values in the When Work Assignments column are based on fields that are populated in the Work Units columns only.
