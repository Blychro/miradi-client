Work Plan View

4A. Develop Detailed Work Plans

Assign Resources

To assign an existing resource to an activity/method or task, on the <Work Plan> tab, select an activity/method or task that you want to focus on. Click on the <Add Resource> button at the side of the page and use the drop down box to select the appropriate human or material resource (do not include financial resources here). You can then specify the number of units for the resource in each upcoming quarter. Repeat the above for all other activities or tasks.

You can add multiple resources to one activity or task if necessary.

You can also assign resources at either the overall activity or task level for a given activity, but you cannot do both.
