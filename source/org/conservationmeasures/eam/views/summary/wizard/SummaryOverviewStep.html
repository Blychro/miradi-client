Intro to Summary View

The window below provides an overview of your project. You can fill in your project's name, start date, and date of last data entry.

Click next to define initial project team.



Navigation Hints for the Summary View and the Overall Application

Summary View

The "General" page contains information fields that most projects will want to fill in. Some organizations will have their own custom data fields for their projects. You can find these by clicking on the appropriate tab.

Overall Application

All views have an interview pane on top and an action pane below. You can resize or hide the interview pane by using the splitter and/or arrows. Use the buttons on the upper right to get context sensitive additional information, examples , and workshop hints.

Note that the program automatically saves all changes so there is no "save" command. You can save a copy under a different name using the "save as" command.
