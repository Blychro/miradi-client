Intro to Summary View

The window below provides an overview of your project. You can fill in your project's name, your project's start and end dates, and the project data effective date (the date of your most recent updating of the Miradi project file).

Click next to define initial project team.



Navigation Hints for the Summary View and the Overall Application

Summary View

The "General" page contains information fields that most projects will want to fill in. Some organizations will have their own custom data fields for their projects. You can find these by clicking on the appropriate tab. The summary page is also used to control several project-wide options such as the number of currency decimal places and the start and end-dates for your project that will affect other views such as workplanning.

Overall Application

All views have an interview pane on top and an action pane below. Use the previous and next buttons to move backwards and forward through the interview. You can resize or hide the interview pane by using the splitter and/or arrows. Use the buttons on the upper right to get context sensitive additional information, examples , and workshop hints.

Note that the program automatically saves all changes so there is no "save" command. You can save a copy under a different name using the "Copy Project To" command.
