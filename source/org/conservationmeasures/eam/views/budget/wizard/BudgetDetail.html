Financial View

4A. Develop Detailed Work Plans

Develop Project Budgets: Budget Detail

At this point, you should have already developed your basic work plan showing activities/methods and tasks and assigning resources to each.

To add budget detail to an activity/method or task, on the <Budget> tab, select the one that you want to focus on. You should already see the number of units for each quarter for that resource. Using the drop down menu, you can then assign an accounting code and funding source for that resource.

Hint: The same resource may have different accounting codes or funding sources for different activities/ methods or tasks.

If you have already added the cost per unit for a resource, at this point you should see the total cost for that resource for the quarter and rolled up for the year. The annual amount is then displayed in the table at the top of the page. Click on the <Resources> tab and select the resource in the table if you need to edit the cost per unit information.

Hint: If you need to adjust the planning horizon for your project, go to the project start and end date fields in the Summary View.

Hint: You can edit the number decimal places for currency amounts in the Summary View.



A resource is a person or item that is needed to complete an activity/method or a task. There are three main kinds of resources:

  • Human Resources: Members of your project team, other members of your organization, partners, advisors, consultants, or other people who can help accomplish tasks. Human resource units are usually measured in some time unit such as hours, days, weeks, or months.

  • Material Resources: Items such as vehicles, boats, fuel, computers, offices, conference rooms, plane tickets, telephones. Material resources are either measured in a time unit (days of boat rental) or as their own unit (numbers of round trip plane tickets).

  • Financial Resources: The money required to pay for most human and material resources. Financial resources are usually figured in one or more standard currencies.
