Welcome to e-Adaptive Management

This software program is being developed by the Conservation Measures Partnership (CMP) to assist conservation practitioners in going through the adaptive management process outlined in the CMP's Open Standards for the Practice of Conservation.

If this is your first time using the program, you will need to create a project, using one of these methods:

  • Create a New Project from scratch

  • Import a Zipped Project

  • Import a TNC CAP Workbook

Or, to continue working on an existing project, select it from the list below.
