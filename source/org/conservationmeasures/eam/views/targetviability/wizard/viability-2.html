Describe Status of Targets

Click on any target in the summary table to display the viability rating details for that target. You can also click on a target in the diagram to see these details.

Miradi currently supports two complementary methods for assessing target status. Click the box to indicate which method you would like to use for this target. You can also set a global preference for your project using the preferences menu.



 
   


Two Methods for Viability Analysis

Miradi currently supports two methods for viability analysis.

  • Basic Viability Analysis - This method asks you to think about each target and to identify specific goals that outline the desired future condition of the target for your project. It then asks you to develop specific indicators that might be used to measure each goal. This method is relatively simple and very flexible, but does not readily lend itself to systematic assesment of target viability. 
  • TNC Style Viability Analysis - This method asks you develop specific key ecological attributes (KEAs) for each target and specific indicators for each KEA. It then has you determine - to the best of your ability - an acceptable range of variation for each indicator, or in other words, to determine what a healthy target would look like. Finally, the method requires that you specify both the current status of each indicator as well as the desired future status (also known as a goal). This method is more systematic and enables comparisons across projects, but is more complex to design and implement.

It is important to note that both methods have the same ultimate outputs - the goal for each target and the indicator that will be used to assess progress towards this goal.
